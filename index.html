What is SQL Murder Mystery

SQL Murder Mystery is a little fun game about solving a murder mystery by only using Structured Query Language (SQL). At the first step you are a confused detective who forgot what is the case about and all you remember are three information which is the date, type of murder, and the location of city where the murder happend. The fun part is that you are give a big chunck of data from crime scene report, interviews, facebook events, and more.

"It feels like becoming a real detective"

/* 3 Suspect */

I'm not going to show the output here since it will ruin the design of the webpage but first we gain the query of each of the clue.

          

/* SUBQUERY */
/* Connect the membership to get person id*/
SELECT person_id FROM get_fit_now_member
WHERE membership_status = 'gold'
AND id IN
(
    SELECT membership_id FROM get_fit_now_check_in
    WHERE membership_id LIKE '48Z%'
);
/* Two suspect */

/* License ID from plate id */
SELECT id FROM drivers_license
WHERE plate_number LIKE '%H42W%';
/* Three suspect */

Connecting the gold status with membership id. Searching drivers_license where the plate number have a subsequence of "H42W"

          
/* CONNECTING THE STRING WITH 3 QUERY */
SELECT * FROM person
WHERE id IN
(
    SELECT person_id FROM get_fit_now_member
    WHERE membership_status = 'gold'
    AND id IN
    (
        SELECT membership_id FROM get_fit_now_check_in
        WHERE membership_id LIKE '48Z%'
    )
)
AND license_id IN
(
    SELECT id FROM drivers_license
    WHERE plate_number LIKE '%H42W%'
);

/* Output */
/*
